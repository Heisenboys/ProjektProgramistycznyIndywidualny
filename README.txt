Projekt jest prostym interfacem dla uzytkownika biblioteki. Biblioteke tworzy baza danych qsqlite z ograniczona liczba tabel oraz danych.
Interface tworzy kilka klas okien bedących klasami pochodnymi QWidget lub QDialog. Klasa bazodanowa to QSqlDatabase, która razem z QSqlQuery wywołuje proste operacje
SELECT, UPDATE, INSERT itp.
